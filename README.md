# Wall Calendar

An interactive wall calendar built with **Next.js** — designed to look and feel like a real calendar hanging on a wall.

## Features

- 📅 Month view with date range selection
- 📝 Per-day notes via right-click context menu
- 🎨 Unique theme color + hero image per month
- 🔄 Page-turn animation when switching months
- 📌 Realistic spiral binding & wall shadow
- 💾 Notes persisted in `localStorage`
- 📱 Fully responsive (mobile + desktop)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Stack

- [Next.js 16](https://nextjs.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- TypeScript
